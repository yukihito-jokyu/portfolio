# プロフィール

はじめまして。Webアプリケーション開発に情熱を注ぐエンジニアです。
ユーザーにとって価値のある体験を実現することを目指しています。

## 技術スタック

- **フロントエンド:**
  - **言語:** TypeScript, JavaScript
  - **フレームワーク/ライブラリ:** React, Redux Toolkit, TanStack Query, Zustand, React Router, TanStack Router
  - **UI/スタイリング:** Tailwind CSS, Material UI (MUI), shadcn/ui, React Flow, Chart.js, React DnD, @floating-ui/react
  - **ユーティリティ:** Axios, i18next, React Markdown, SweetAlert2, Zod
  - **ビルドツール:** Vite
- **バックエンド:**
  - **TypeScript (Node.js):**
    - **プロトコル:** Model Context Protocol (MCP)
    - **ライブラリ:** @modelcontextprotocol/sdk, mysql2, commander, zod
  - **Go:**
    - **フレームワーク:** Echo, Gin
    - **ORM/DB:** GORM
    - **ライブラリ:** Watermill, Google GenAI SDK, JWT, UUID, Testify, Goose
  - **Python:**
    - **フレームワーク:** FastAPI, Flask
    - **ORM/DB:** SQLAlchemy
    - **ライブラリ:** Pydantic, TKinter, PySimpleGUI
  - **データベース/インフラ:** MySQL, PostgreSQL, SQLite, Firebase (Admin SDK)
- **データサイエンス/AI:**
  - **ライブラリ:** PyTorch, Torchvision, Pandas, NumPy, Scikit-learn, SciPy, Seaborn, Matplotlib, OpenCV, OpenAI Gym, Pygame, Pillow, Tqdm, TensorFlow, Keras, MeCab, spaCy, XGBoost, LightGBM, wisper, VOICEVOX
  - **AIサービス:** Google Generative AI (Gemini)
- **開発ツール/インフラ:**
  - **ツール:** Git, Docker, Docker Compose, VS Code, Jupyter Notebook
  - **デプロイ/ホスティング:** Vercel, Cloudflare, github pages
  - **パッケージ管理:** pnpm, npm, Rye, uv

## 経歴

小規模な検証プロジェクトから大規模なWebアプリケーションまで、幅広いプロジェクト開発に携わってきました。
単にコードを書くだけでなく、課題解決や新しい技術の習得にも日々積極的に取り組んでいます。
