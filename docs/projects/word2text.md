# word2text

## 概要
**開発時期**: 2024年4月 - 2024年6月

『ゼロから作るDeep Learning 2』を参考に実装された、単語の分散表現（Word Embedding）に関連する実験的なプロジェクトです。
自然言語処理（NLP）の基礎的な技術を検証し、データセットの処理方法がモデルの精度に与える影響を実験しています。

## 主な機能
- **英語データセットの学習**: 翻訳を行わず、英語のままのデータセットを用いてモデルを学習させています。
- **実験結果**: データセットに含まれる単語に対しては高い精度を示す一方、未知語（データセットに含まれない単語）に対しては脆弱であるという特性を確認しました。
