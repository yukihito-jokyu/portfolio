# 制作実績

これまでの制作実績の一部をご紹介します。各プロジェクトの詳細ページで、技術的な解説や動作確認をご覧いただけます。

## [chat-branch](/projects/chat-branch)

会話の「深掘り」と「本筋の維持」を両立させる、階層型チャットアプリケーションです。メッセージからサブチャットを分岐させ、議論の結果を親チャットに統合できます。

![chat-branch](/projects/chat-branch/image.png)

## [DLab](/projects/DLab)

「ノーコードで深層学習を」をテーマに、中高生やプログラミング未経験者でも直感的にAIモデルの構築から学習までを体験できるプラットフォームです。
深層学習の敷居を下げ、次世代のAIエンジニア育成に貢献することを目指しています。

![DLab logo](/projects/DLab/image.png)

## [SimpLyz](/projects/SimpLyz)

「SimpLyz」は、データ分析の初学者が手軽にデータサイエンスを学び、実践できるノーコードWebアプリケーションです。
直感的な操作で、データの可視化や前処理、モデル作成までをスムーズに行えます。

![SimpLyz](/projects/SimpLyz/image.png)

## [word2text](/projects/word2text)

『ゼロから作るDeep Learning 2』を参考に実装した、単語からテキストを生成する実験的なプロジェクトです。
自然言語処理の基礎を学び、モデルの挙動を検証しました。

![word2text logo](/projects/word2text/good.png)

## [MySQL MCP Server](/projects/mysql-mcp-server)

バックエンド開発の安全性を重視した、読み取り専用のMySQL Model Context Protocol (MCP) サーバーです。誤操作によるデータ破損を防ぎながら、スキーマ確認やデータ参照を効率化します。
