プロンプト集に新しいコンテンツを追加する。

以下の手順で対話形式で進めること。

## 手順

1. ユーザーに以下の項目を順に質問し、回答を受け取る:
   - プロンプト名（日本語のプロンプト名からkebab-caseにAIが変換する）
   - 用途（どのような場面で使うか）
   - プロンプト本文
   - 入力例(AIが考える)
   - 出力例(AIが考える)

2. 回答が揃ったら、以下のファイルを作成・更新する:

   ### a) 詳細ページの作成

   `docs/prompts/{プロンプト名}.md` を以下のテンプレートで作成:

   ```markdown
   # {プロンプト名}

   ## 用途

   {用途}

   ## プロンプト
   ```

   {プロンプト本文}

   ```

   ## 入力例

   ```

   {入力例}

   ```

   ## 出力例

   ```

   {出力例}

   ```

   ```

   ### b) 一覧ページの更新

   `docs/prompts.md` の末尾に以下を追記:

   ```markdown
   ## [{プロンプト名}](/prompts/{プロンプト名})

   {用途}
   ```

   ### c) サイドバー設定の更新

   `docs/.vitepress/config.mts` の sidebar 内「プロンプト集」に items を追加（なければ items 配列を新規作成）:

   ```typescript
   {
     text: "プロンプト集",
     link: "/prompts",
     items: [
       { text: "{プロンプト名}", link: "/prompts/{プロンプト名}" },
     ],
     collapsed: true,
   }
   ```

   既に items が存在する場合は、配列の末尾に `{ text: "{プロンプト名}", link: "/prompts/{プロンプト名}" }` を追加する。

3. 全てのファイル作成・更新が完了したら、実行結果をユーザーに報告する。
